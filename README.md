# Lea's Spaceship

This is Lea's spaceship. This is a submission to the spaceship challenge at <https://threejs-journey.com>.

## Notable stuff

I'm using a combination of sines and cosines to create the sky pattern. A playground for that cheap noise function is here:

<https://codepen.io/learosema/pen/mdWJYJM?editors=1010>

Multiple scales of that noise form an interesting pattern.
This is loosely inspired by the technique described in the book about shaders, called Fractal brownian motion: <https://thebookofshaders.com/13/>.
